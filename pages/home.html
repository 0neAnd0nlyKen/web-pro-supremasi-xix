Tentang Sistem FLISH FLASH

FLISH FLASH merupakan sebuah platform berbasis web yang dirancang untuk menampilkan berbagai permainan flash dalam bentuk tampilan video pendek seperti pada platform media sosial modern. Sistem ini memungkinkan pengguna untuk menemukan, menonton, dan memainkan berbagai game secara lebih interaktif dan mudah diakses.

Dengan konsep streaming short video, pengguna dapat menjelajahi berbagai permainan hanya dengan melakukan scroll pada halaman utama sehingga pengalaman bermain menjadi lebih cepat, menarik, dan efisien.

Informasi Sistem

Platform Web

Sistem dikembangkan menggunakan HTML, CSS, JavaScript, dan Bootstrap untuk menghasilkan tampilan yang responsif dan modern.

Game Flash Engine

Sistem menggunakan teknologi Ruffle Emulator untuk menjalankan game Flash langsung pada browser modern tanpa plugin tambahan.

Single Page Application

Website ini menggunakan konsep SPA sehingga navigasi halaman dapat dilakukan dengan cepat tanpa perlu reload halaman.

Cara Kerja Sistem

  1. Pengguna membuka website FLISH FLASH melalui browser.
  2. Sistem menampilkan halaman utama yang berisi game unggulan.
  3. Pengguna dapat menjelajahi game lain dengan melakukan scroll.
  4. Game dapat dimainkan langsung menggunakan emulator Flash.
  5. Pengguna dapat memberikan like atau mengikuti challenge.
