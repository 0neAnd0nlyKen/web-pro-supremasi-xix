Apakah Sobat berinspirasi menyusun game tetapi berasa bingung mau menyusun apa dulu? Anda bisa menggunakan aplikasi web ini untuk
melihat kembali game-game Flash yang dari dulu sudah membawa inspirasi ke banyak orang! Gunakan UI terinspirasi platform short video untuk melihat dan 
memainkan game dengan sekilat. Dan kalupun tiidak tertarik bisa dengan mudah scroll untuk mendapat game berikutnya! Jelajahilah sebuah era historis yaitu
Flash untuk mendapat inspirasi, kemudian ujilah diri dengan berbagai challenges yang pastinya relevan dengan game interest Anda!

Aktifkan index.html pada sebuah port dengan berbagai metode, misalnya sebagai berikut:

1. Membuka deployment serverless Vercel pada link berikut:

```
web-pro-supremasi-xix.vercel.app/
```

2. Menggunakan Python HTTP server:

```
python -m http.server 8000
```

3. Menggunakan VS Code Live Server Extension yang bisa ditemukan pada link berikut:

```
https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer
```
